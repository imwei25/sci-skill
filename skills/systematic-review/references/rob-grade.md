# 偏倚风险评估（RoB）+ GRADE 证据分级

> 这两步是**人类方法学裁决**，本文件给的是规则化的判定标准与算法，不是替你下结论。逐条有据可依。

## 一、偏倚风险工具（按研究设计选，别乱套）
| 研究设计 | 工具 | 评估维度（domain） |
|---|---|---|
| 随机对照试验 | **RoB 2** | ①随机化过程 ②偏离既定干预 ③结局数据缺失 ④结局测量 ⑤选择性报告 |
| 非随机干预研究 | **ROBINS-I** | ①混杂 ②对象选择 ③干预分类 ④偏离干预 ⑤缺失数据 ⑥结局测量 ⑦选择性报告 |
| 队列/病例对照（观察性） | **NOS**（纽卡斯尔-渥太华，星级） | 选择(4★)/可比性(2★)/暴露或结局(3★) |
| 诊断准确性 | **QUADAS-2** | ①病例选择 ②待评试验 ③金标准 ④流程与时间；前 3 项另评适用性 |
| 患病率/现况 | **JBI** 现况研究清单 | 抽样框、抽样方式、样本量、对象与场所描述、覆盖度、测量效度信度、统计 |

## 二、判定机制（以 RoB 2 为范式）
1. 每个 domain 有一组 **signaling questions**，逐条回答：**Y（是）/ PY（可能是）/ PN（可能否）/ N（否）/ NI（信息不足）**。
2. 按工具的**判定算法**把该 domain 的回答映射成 **Low / Some concerns / High**（RoB 2）或 **Low / Moderate / Serious / Critical / No information**（ROBINS-I）。
   - 例（RoB 2 结局缺失域）：数据基本完整或缺失不依赖真实值 → Low；缺失可能与真实值相关且无稳健敏感性分析 → High；介于之间 → Some concerns。
3. **总体偏倚 = 最差 domain 拖底**（RoB 2：任一 High → 总体 High；有 Some concerns 但无 High → 总体 Some concerns；全 Low → Low）。ROBINS-I 同理按最严重域定总体。
4. **NI（信息不足）不等于 Low**——报告不清就不能给低风险，倾向 Some concerns/记"信息不足"并可联系作者。

### RoB 2 五域核心 signaling questions（速查；完整版以官方 RoB 2 工具/Excel 为准）
每问按 Y/PY/PN/N/NI 答，再按官方算法映射该域 Low/Some concerns/High：
- **D1 随机化过程**：分配序列是否真随机产生？分配是否隐藏（至入组）？基线组间差异是否提示随机化有问题？
- **D2 偏离既定干预**：受试者/施治者是否对分组设盲？有无影响结局的非方案偏离？是否按 **ITT（意向性治疗）**分析？
- **D3 结局数据缺失**：结局数据是否基本完整（或缺失少且组间相近）？缺失是否可能依赖真实值？有无稳健的敏感性分析支撑结果？
- **D4 结局测量**：测量方法是否恰当且组间一致？结局评定者是否对分组设盲？知晓分组是否可能影响测量（**主观结局尤其要警惕**）？
- **D5 选择性报告**：是否有预注册/方案？报告的结局与分析是否与计划一致？有无"按结果择优报告"迹象？
> 非随机干预用 ROBINS-I（7 域，多一个"混杂"前置域）、观察性用 NOS 星级、诊断用 QUADAS-2——signaling questions 各不同，同样以各自官方工具为准。

## 三、RoB 产出
- 每个研究一行、各 domain 一列 + 总体列的表；配 `nature-figure` 的 **traffic-light 图**（绿/黄/红）与 **summary 条形图**。
- 判定理由**逐 domain 写一句依据**（引原文位置），别只给颜色。

## 四、GRADE 证据确定性分级（对每个主要结局分别评）
终点的证据质量分四级：**高 / 中 / 低 / 极低**。起评级别看设计：**RCT 从"高"起评**，**观察性研究从"低"起评**。

### 五个降级因素（每个可降 1–2 级）
1. **偏倚风险（Risk of bias）**：纳入研究整体 RoB 高 → 降。
2. **不一致性（Inconsistency）**：结果异质性大（**I² 高**、可信区间重叠差、点估计方向不一）且无法用亚组解释 → 降。（**I²/τ² 来自第 7 步 Meta 合并**——先做定量合成再评本域；只纳入 1 项研究时本域不适用。）
3. **间接性（Indirectness）**：人群/干预/对照/结局与问题不完全对应（如用替代结局、间接比较）→ 降。
4. **不精确性（Imprecision）**：可信区间宽、跨越无效线或临床决策阈值、样本量/事件数不达最优信息量（OIS）→ 降。
5. **发表偏倚（Publication bias）**：漏斗图不对称、Egger 检验阳性、只见小阳性研究、疑有未发表阴性结果 → 降。（⚠️ **研究数 <10 时漏斗图/Egger 不可靠，别硬做**——改凭"是否只见小样本阳性研究、有无注册却未发表的试验、是否行业资助"等间接线索判断；实在无法判断就记"不可评估"，别默认不降也别硬降。）

### 三个升级因素（仅用于**观察性研究**、且无严重降级时）
1. **大效应量**：RR/OR 很大（如 >2 或 <0.5，且无明显混杂）→ 升 1 级；极大（>5 或 <0.2）→ 升 2 级。
2. **剂量-反应关系**：存在梯度 → 升。
3. **混杂方向**：所有可能的残余混杂都会**削弱**观察到的效应（即真实效应只会更强）→ 升。

### 判定流程
起评级别 → 逐一评五个降级域、累计降级步数 → （观察性且未严重降级时）评三个升级因素 → 得最终级别（不低于"极低"）。**每一步降/升都要写明理由**。产出 **Summary of Findings（SoF）表**：每结局一行，列出研究数/受试数、效应量+95%CI、证据确定性(GRADE)、以及降/升级注脚。
