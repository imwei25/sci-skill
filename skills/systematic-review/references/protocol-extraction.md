# PROSPERO 协议字段 + 数据提取 + 效应量换算

## 一、PROSPERO 注册字段清单（筛文献前离线备齐，再录网页表单）
> PROSPERO 收录健康相关系统综述协议。**先注册后筛选**是系统综述信誉的底线。逐项备好，缺项别留空硬交。

- **综述标题**（含 PICO 要素、综述类型）
- **检索时间范围 / 计划检索日期**
- **研究团队与单位、通讯人**
- **资助来源与利益冲突**
- **综述问题（PICO/PECO）**：Population / Intervention(Exposure) / Comparator / Outcome
- **纳入标准**：人群、干预/暴露、对照、结局、研究设计、语言、发表状态
- **排除标准**（与纳入互补、可操作）
- **信息来源**：拟检索的数据库与平台、试验注册库、灰色文献、手工检索/引文追踪
- **检索策略**（至少一个库的完整检索式，或说明将在附录提供）
- **研究筛选流程**：几人筛、独立与否、冲突消解办法
- **数据提取流程**：几人提、独立与否、用什么表
- **偏倚风险评估**：用哪个工具、几人评
- **主要结局 / 次要结局**（预先定义，防事后挑）
- **数据合并计划**：是否做 Meta、效应量类型、固定/随机效应、异质性评估(I²)、亚组/敏感性分析、发表偏倚评估
- **证据分级**：是否用 GRADE
- **预计开始/完成日期、当前阶段**

⚠️ 注册后对**主要结局、纳排、主分析**的改动属可见偏离，须在协议更新与最终论文里如实说明（与 `research-design` 的偏离日志同源）。

## 二、数据提取（预定义 schema，双人独立提最稳）
**提取字段建议**（按综述问题裁剪）：
- 研究标识：作者、年份、国家、设计、注册号
- 人群：样本量、年龄、性别、关键基线、纳排要点
- 干预/暴露与对照：具体方案、剂量、疗程
- 结局：定义、测量时点、测量工具
- 结果数据：各组 n、事件数或均值±SD（或中位数[IQR]）、效应量+95%CI、随访时长
- 偏倚风险各 domain 判定
- 备注：缺数据/需联系作者/换算过程

**规矩**：通读全文后提取；**关键数据缺失先联系通讯作者**再决定取舍；只从图里读数（用取点工具）要标注不确定性；双人提取不一致处记录并消解。

## 三、效应量换算公式（统一口径、为 Meta 合并做准备）
纳入研究报告口径常不一致，合并前先换算到同一效应量。常用换算：

- **中位数[IQR] → 均值±SD**（近似，适用于近对称分布）：
  - 均值 ≈ 中位数；SD ≈ IQR / 1.35（即 (Q3−Q1)/1.35）。
  - 报"中位数(min–max, n)"时用 Wan et al. 2014 或 Hozo et al. 2005 的公式（按 n 选系数），如实标注为近似。
- **SE ↔ SD**：SD = SE × √n；SE = SD / √n。
- **95%CI → SD**（连续结局，均值差）：SD = √n × (上限 − 下限) / (2 × 1.96)（大样本 z；小样本用对应 t 值替 1.96）。
- **95%CI → SE**（比值型 OR/RR/HR，在对数尺度上算）：SE(lnEffect) = (ln上限 − ln下限) / (2 × 1.96)。
- **OR → RR**（需基线风险 p0）：RR = OR / (1 − p0 + p0 × OR)。低发生率时 OR≈RR，高发生率时差异大，别直接混用。
- **p 值 → SE / z**（无 CI 时的下策）：由精确 p 反推 z，再 SE = 效应量 / z；仅在拿不到 CI 时用，标注局限。
- **多臂研究共享对照**：合并时按 Cochrane Handbook 处理（拆分共享对照组样本，避免重复计数）。

> 换算都是近似，**优先向作者要原始数据**；换算过程记进提取表备注，可复核。定量合并（森林图、I²/τ²、随机效应 DerSimonian-Laird、亚组、敏感性）交 `data-analysis`（`statsmodels.stats.meta_analysis`）。⚠️ **statsmodels 不含**：Egger 发表偏倚检验/漏斗图不对称、REML、网络 Meta/多水平/剂量-反应——这些用 R `metafor`/`netmeta`，别声称 Python 能做；如实告知用户。
