# 深度研究技能

对一个**足够具体**的问题，产出一份多来源、可核查、带引用的报告。**跨框架通用**（OpenCode / OpenClaw / WorkBuddy / Claude Code 等，不绑定任何具体框架或模型）。**由本技能自行判断该并行调研还是串行调研**（见「编排决策」）。

## Python 环境（用于抓文献/网页）
> 没有项目根 `.venv`？先运行 `env-setup` 技能建好并装依赖。
```
.venv/Scripts/python.exe   # Windows（正斜杠写法，bash 与 PowerShell 都能用）
.venv/bin/python           # Linux / macOS
```
可用 `requests` / `beautifulsoup4` 抓取解析。

## 流程
1. **收敛问题**：太宽就先问 2-3 个澄清问题（范围、时间、地域、用途），把问题缩到可研究。
2. **拆子问题**：列出 3-6 个需要回答的关键子问题。
3. **编排决策：并行还是串行？**（拆完子问题后，先自问一次再动手取证）
   - 先判断**运行时能力**：当前环境是否能开**子代理**（任务派发/spawn 类工具）？注意"能并发调用工具"≠"能开子代理"——有的框架能并发发起工具调用但没有子代理机制。**工具列表里没有子代理/任务类工具，或拿不准 → 一律串行**，跳到「串行」。
   - 再按**复杂度分档**（借鉴 Anthropic 编排经验，避免过度并行）：
     - **简单**（1-2 个子问题、纯事实核对、来源少）→ **串行**，不开子代理，主流程自己顺序查完。
     - **中等**（3-4 个**相对独立**的子问题）→ **并行**：每个子问题派一个子代理并发取证，各自独立上下文、明确目标与边界。
     - **复杂**（5+ 子问题，或需大量抓取/多来源交叉）→ **并行**，明确分工，主代理只负责综合。
   - **子问题之间有依赖**（后一个要用到前一个的结论）→ 该段**串行**；无依赖的部分才并行。
   - 并行铁律：给每个子代理**明确的目标、期望输出格式、来源类型、任务边界**（别只说"查 X"）；宁可少开几个也不要为简单问题铺满子代理；子代理只回传结论与来源，主代理负责去重与综合。
4. **多源取证**（每个子问题至少 2 个独立来源；按上一步决定并行/串行执行）：
   - 学术证据：**默认用 `literature-review` 的 `search.py`（走 Europe PMC，国内可达、无人工审批门）**；`search-lit` 作为有 claude.ai MCP 或境外网络时的增强路径。
   - 网络证据：有 web 搜索/抓取能力时取权威来源（官方文档、指南、统计机构）。
   - **来源分级**：临床指南 / 系统综述 > RCT > 官方统计 > 一般同行评议论文 > 权威媒体 > 博客/预印本。取证与写作时优先高等级来源，参考清单里标注每条的等级。
   - **检索日志**（可复现的最低要求）：逐条记录 子问题 | 检索式 | 来源 | 日期 | 关键结论（标题 + URL/DOI）。
   - **停止标准**：每个子问题取到 2–4 个独立来源、或连续两轮无新信息即停，避免无限查下去。
5. **交叉核实**：对关键论断，看多个来源是否一致；有冲突就并列呈现并标注分歧，别只挑一个。
6. **反向核查（成稿前一轮）**：挑 3–5 条最关键结论，**反过来找反证**（用否定式/竞争假说去检索），能扛住反驳的才写成结论，扛不住的降级到"不确定"并入局限。
7. **综合成报告**，写到 `outputs/deep_research.md`；速览结论每条标**置信度（高/中/低 + 一句理由）**。

## 报告结构
- **问题与范围**
- **速览结论**（3-5 条要点，每条带引用）
- **分子问题详述**（证据 + 出处 + 一致性/分歧）
- **不确定性与局限**（哪些没查实、来源质量如何）
- **参考来源清单**（编号，正文用 [n] 对应）

## 约定
- 产出写 `outputs/deep_research.md`。
- **出 PDF（正式报告默认交付版）**：报告写完后，把 `outputs/deep_research.md` 交给 `render-pdf-doc` 技能渲染成 `outputs/deep_research.pdf`。中文报告务必指定中文字体（`--cjk-font`：本地 `Microsoft YaHei`，服务器 `Noto Sans CJK SC`），否则会漏字。
- **每个关键论断都要有出处**；查不到就明说"未找到可靠来源"，不要脑补。
- 区分事实、推断、观点；对相互矛盾的证据如实并列。
- 联网受限时说明这限制了结论强度。
