# 预注册草案——模板与偏离日志

> 预注册 = 在**看结局数据之前**把假设、主要结局和主分析计划写死。注册后再改，就是**可见的偏离**（如实披露即可），而不是藏在自由度里的隐形 p-hacking。防 HARKing（看到结果再编假设）。
>
> **缺的真实信息向用户要，绝不替编**：注册号、伦理批号、样本量参数（应来自预实验或文献）。

## 去哪注册（据研究类型）
| 类型 | 平台 | 时点 |
|---|---|---|
| 临床试验（干预性） | **ChiCTR**（中国）/ ClinicalTrials.gov | **入组前**（多数期刊硬性要求，ICMJE） |
| 系统综述 / Meta | **PROSPERO** | **筛文献前** |
| 一般 / 观察性 / 假设 | **OSF**（osf.io） | 采数/看结局前 |

## 预注册草案模板
```markdown
# 预注册：<研究简称>
**日期：** YYYY-MM-DD    **拟注册平台：** ChiCTR / ClinicalTrials.gov / PROSPERO / OSF
**注册号：** [提交后回填，注册前留空——不编]

## 1. 背景与理据（≤200 字）
[为什么做；引 1–3 篇真实文献；接新颖性裁定记录的差异点陈述]

## 2. 主假设（可证伪）
H1: ...
（如有）H0: ...

## 3. 设计
- 研究类型：RCT / 队列 / 病例对照 / 横断面 / 诊断试验 / 系统综述 ...
- 对应报告规范：CONSORT / STROBE / STARD / PRISMA ...

## 4. 对象
- 纳入标准：...
- 排除标准：...
- 抽样 / 分组（随机化方法、分配隐藏、盲法——如适用）：...

## 5. 变量
- 暴露 / 干预：...    对照：...
- **主要结局**（单一、预先定义、含测量时点与方法）：...
- 次要结局：...
- 拟测量的协变量 / 混杂：...

## 6. 样本量 / 把握度（先验）
- 参数（差值/SD/率/HR，来自预实验或文献 [引]）：...
- 用 clinical-stats 估算：α=0.05 双侧，power=0.8，脱落率=___ → 每组 n=___
- ⚠️ 不做事后功效分析。

## 7. 统计分析计划
- 主要结局分析：<检验/模型>，协变量：...
- **预先声明的**亚组 / 交互分析（其余一律标"探索性"）：...
- 多重比较处理：Bonferroni / Benjamini–Hochberg（多终点）
- 缺失数据处理：<完整病例 / 多重插补 m=__>
- 软件及版本：...

## 8. 伦理
- 审批机构 / 批号：[真实，缺则"待补充"]
- 知情同意：...
```

## 偏离日志（注册后任何改动都记这里，投稿时披露）
```markdown
## 偏离日志
| 日期 | 原计划 | 实际改为 | 原因 | 影响（对主结论） |
|---|---|---|---|---|
| | | | | |
```
- 主要结局、主分析方法、纳排的改动属**重大偏离**，必须记录并在论文 Methods/Limitations 里说明。
- **绝不用观察到的结果反向修改主假设或主要结局**——那是 HARKing，不是偏离披露能洗白的。
